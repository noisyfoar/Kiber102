ИИ чат-бот “ИИ сонник” 
Необходимо разработать ИИ чат-бота для интерпретации снов, состоящий из веб-страницы с окном для ведения диалога по заданному сценарию, с подключением LLM и настройкой системных промптов.
Контекст: сны - это способ подсознания поговорить с тобой.
Интерпретация снов - современный и мощный инструмент самопознания и самоанализа.
Уклон в психологическую, не эзотерическую интерпретацию.
Требуется: наличие приятного тематического UI фронтенда реализация БД для сохранения данных о пользователе (имя, дата рождения, хранение контекста запросов) для учета при повторных запросах Реализация ASR для распознавания речи пользователя и TTS для озвучивания ответов сонника
Оценивается: максимальная приближенность к MVP, который можно запустить на тестовую аудиторию и куда пользователь захочет вернуться: наличие логичного эмпатийного пользовательского флоу, персонализированное общение с пользователем.
Огромным плюсом будет: реализация чат-бота в Телеграмм с заданной логикой (в дополнению к веб-странице) подключение модуля оплаты Робокасса (API предоставим) авторизация по введенному номеру телефона (пока без OTP) и сохранение контекста прошлых сессий для выдачи результатов в динамике на базе нескольких последних запросов реализация “эмпатичного” дерева диалогов для выдачи максимально полезной и релевантной интерпретации.
